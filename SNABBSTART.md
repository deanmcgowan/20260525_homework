# Snabbstartsguide - Stormaktstiden PWA

## 🚀 Kom igång på 2 minuter

### Steg 1: Öppna appen
```bash
# I terminalen (valfritt):
cd /path/to/20260525_homework
python -m http.server 8000

# Eller använd npx:
npx serve
```

Sedan öppna i webbläsare: `http://localhost:8000`

**För iPhone 12:**
- Öppna Safari och navigera till appen
- Tryck på dela-knappen
- Välj "Lägg till på hemskärmen"
- Nu fungerar den som en native app!

### Steg 2: Börja lära dig!

#### Rekommenderad studieplan

**Dag 1-2: Grundläggande kunskap**
1. **Kungens Kort** (4-6 min) - Lär känna de 5 kungarna
2. **Snabbquiz** (3-5 min) - Testa grundläggande fakta
3. **Tidslinjen** (5-7 min) - Förstå kronologin

**Dag 3-4: Fördjupning**
4. **Berättelse** (7-10 min) - Se helheten och samband
5. **Ljudberättelse** (5-10 min) - Lyssna och lär
6. **Rita & Anteckna** (5-8 min) - Skapa visuella minnesbilder

**Dag 5-6: Repetition**
7. **Snabbmatchning** (3-4 min) - Snabb repetition
8. **Snabbquiz** (3-5 min) - Testa igen
9. **Kungens Kort** (4-6 min) - Repetera svåra kungar

**Dag 7: Provtest**
10. **Provtest** (10-15 min) - Simulera det riktiga provet
11. Granska facit
12. **Snabbquiz** för saker du missade

### Steg 3: Följ din framsteg

Tryck på **"📈 Visa Framsteg"** för att se:
- Dina stjärnor ⭐
- Din streak 🔥
- Träffsäkerhet 📊
- Behärskning per kung 👑
- Upplåsta achievements 🏆

## 💡 Tips för bästa resultat

### För elever med ADHD:

**✅ GÖR:**
- Sätt en timer för varje session (max 15 min)
- Ta en 5-minuters paus mellan sessionerna
- Använd olika lägen - variation är nyckeln!
- Fira varje liten framgång (stjärnor, achievements)
- Studera vid samma tid varje dag för att bygga rutin

**❌ UNDVIK:**
- Långa sessioner utan paus
- Samma läge flera gånger i rad
- Studera när du är trött eller hungrig
- Hoppa över "lätta" frågor - repetition hjälper!

### Inför provet:

**3 dagar innan:**
- Genomför Provtest-läget
- Identifiera svaga områden
- Fokusera på dessa med Kungens Kort och Snabbquiz

**1 dag innan:**
- Rita tidslinje från minnet
- Genomför ett sista Provtest
- Lyssna på Ljudberättelser för att repetera

**På provdagen:**
- Snabbquiz på morgonen (5 min)
- Känn dig trygg - du har tränat!

## 🆘 Felsökning

**Appen laddar inte:**
- Kontrollera internetanslutning (första gången)
- Rensa cache och ladda om
- Prova en annan webbläsare

**Ljudberättelse fungerar inte:**
- Kontrollera att volym är på
- Vissa webbläsare stöder inte Text-to-Speech
- Prova Safari på iOS för bäst resultat

**Framsteg sparas inte:**
- Kontrollera att cookies/storage är aktiverat
- Använd inte privat/inkognito läge
- IndexedDB måste vara tillåtet

**Touch fungerar inte:**
- Kontrollera att JavaScript är aktiverat
- Ladda om sidan
- Testa i Safari på iOS

## 📞 Support

Om något inte fungerar:
1. Öppna webbläsarens konsol (F12)
2. Leta efter felmeddelanden
3. Prova `window.debugApp.resetProgress()` för att börja om
